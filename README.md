# CNN 2D para Classificação de Mãos (Palma vs Dorso)

Pequeno projeto autodidata desenvolvido em **Python** com **PyTorch** para classificação de imagens de mãos em duas categorias: **palma** e **dorso**.

## Objetivo
Explorar conceitos de **Redes Neurais Convolucionais (CNNs)** aplicados à visão computacional, servindo como etapa inicial de aprendizado para o projeto de pesquisa **BioSense+**.

## Sobre o BioSense+
O [**BioSense+**](https://github.com/vbl10/biosense) é um projeto em desenvolvimento focado na criação de uma plataforma de apoio ao diagnóstico médico de enfermidades **palmares e plantares**, utilizando técnicas de inteligência artificial.

## Tecnologias
- Python
- PyTorch

## Funcionalidades
- Pré-processamento de imagens de mãos
- Treinamento de modelo CNN 2D
- Classificação binária: palma vs dorso
- Avaliação de desempenho do modelo
